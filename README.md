# 每日 TodoList

集成企业微信通知的每日待办管理工具。

## 功能特性

- **Todo 管理**：增删改查待办事项，支持按日期查看
- **完成状态**：点击勾选完成/取消完成
- **企业微信通知**：
  - 每天 **9:30** 发送早间提醒，提醒规划今日任务
  - 每天 **17:30** 发送日终总结，展示完成情况
- **任务累积**：未完成的任务自动累积到第二天，标记 `[累积]` 标签
- **进度展示**：实时显示任务完成进度

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务

```bash
npm start
```

服务默认运行在 `http://localhost:3000`

### 3. 配置企业微信 Webhook

1. 在企业微信群聊中添加「群机器人」
2. 复制机器人的 Webhook 地址
3. 打开 TodoList 页面 → 点击「设置」→ 粘贴 Webhook 地址 → 保存
4. 点击「测试早间通知」或「测试日终通知」验证配置

## 通知逻辑

| 时间 | 动作 | 说明 |
|------|------|------|
| 09:30 | 早间提醒 | 自动累积昨日未完成任务，提醒规划今日 todo |
| 17:30 | 日终总结 | 统计完成/未完成任务，提醒未完成任务将累积到明天 |

## 技术栈

- **后端**：Node.js + Express
- **数据库**：SQLite (better-sqlite3)
- **定时任务**：node-cron
- **通知**：企业微信群机器人 Webhook
- **前端**：原生 HTML/CSS/JS（无框架依赖）
